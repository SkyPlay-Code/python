## üöÄ Installation & Setup

### 1. Prerequisites
You will need **FFmpeg** installed on your system for media conversions.
*   **Windows:** `choco install ffmpeg`
*   **Mac:** `brew install ffmpeg`
*   **Linux:** `sudo apt install ffmpeg`

### 2. Python Environment
```bash
git clone https://github.com/SkyPlay-Code/python.git
cd python
pip install -r requirements.txt
```

**Key Dependencies included:**
`opencv-python`, `mediapipe`, `ultralytics` (YOLOv8), `google-generativeai`, `PyMuPDF`, `aiohttp`, `yt-dlp`.

### 3. API Configuration
For AI features, set your API key:
```bash
export GEMINI_API_KEY="your_key_here"
```
